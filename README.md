# 📄 Smart ATS Resume Analyzer

A powerful AI-driven ATS Resume Analyzer that compares your resume against a job description and provides match percentage, skill gaps, and improvement suggestions.

## 🌟 Features
✅ Upload PDF resumes  
✅ Compare with Job Descriptions  
✅ AI-powered Analysis (Match %, Skill Gaps, Suggestions)  
✅ Downloadable Report  
✅ Powered by Google Gemini API  

## 🛠 Tech Stack
- **Frontend**: Streamlit  
- **AI Model**: Google Gemini 1.5 Flash  
- **Backend**: Python  
- **PDF Parsing**: PyMuPDF  
- **Deployment**: Streamlit Cloud  

## 🚀 Installation & Setup
1. Clone the repo
```bash
   git clone https://github.com/YOURUSERNAME/ai-resume-analyzer.git
   cd ai-resume-analyzer
```
2. Install Dependencies
```bash
   pip install -r requirements.txt
```
3. Create a `.env` file and add
GOOGLE_API_KEY=your_api_key_here

PROMPT=your_prompt_here
4. Run locally
```bash
   streamlit run app.py
```
